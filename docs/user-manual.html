시스템 개요

EGTRONICS COMS(Charger Operation Management System)는 EV 충전기의 생산요청 → 생산착수 → 생산완료 → 출하 → AS 전 과정을 한 곳에서 관리하는 사내 업무 시스템입니다.

빌드 없이 브라우저에서 바로 동작하는 웹앱으로, PC와 태블릿·모바일 모두에서 사용할 수 있습니다. 사무실과 현장(태블릿)을 오가며 쓰는 것을 고려해 터치 조작에 맞춰 설계되어 있습니다. 로그인한 계정의 역할에 따라 접근 가능한 화면이 자동으로 조정됩니다.

ℹ️
DB상 상태값은 4가지뿐이지만 업무 단계는 생산완료와 출하대기로 나뉩니다. 둘 다 AWAIT_PICKUP 상태이며, 발주처·납품정보(영업정보) 입력이 끝났는지 여부로만 구분됩니다. 자세한 내용은 오더 상태 흐름을 참고하세요.

역할 및 권한

COMS는 4개 역할(관리자·영업·생산·품질)을 지원합니다. 별도의 "AS 담당자" 역할은 없으며, AS 접수·처리 화면은 관리자·영업·품질 역할에게 열려 있습니다. 로그인 후 역할에 따라 상단 탭 메뉴가 자동으로 조정됩니다.

역할 부서 예시 접근 가능 화면
관리자 충전기개발실 대시보드, 생산요청, 생산 대기, 출하대기, 통합 조회, AS 접수, AS 처리, 사용자 관리 (전체)
영업 영업본부 대시보드, 생산요청, 생산 대기, 통합 조회, AS 접수, AS 처리
생산 제조생산본부 대시보드, 생산 대기, 출하대기, 통합 조회
품질 품질관리본부 대시보드, 출하대기, 통합 조회, AS 접수, AS 처리
💡
모든 역할이 대시보드와 통합 조회에는 접근할 수 있습니다. 목록에 없는 화면으로 직접 URL·상태를 이동해도 조회 화면으로 대체 표시됩니다.

로그인

COMS는 아이디(ID)와 비밀번호로 로그인합니다. 계정은 사용자 관리 화면(관리자 전용)에서 발급합니다.

1
아이디 입력
관리자로부터 발급받은 아이디를 입력합니다.
2
비밀번호 입력 후 로그인
비밀번호를 입력하고 로그인 버튼을 누릅니다. 확인하는 동안 버튼이 확인 중…으로 바뀝니다.
3
역할별 시작 화면으로 이동
로그인 성공 시 역할에 맞는 대시보드로 자동 이동됩니다. 로그인 정보는 새로고침해도 유지됩니다.

아이디·비밀번호를 비운 채 로그인하면 아이디와 비밀번호를 입력하세요, 잘못 입력하면 아이디 또는 비밀번호가 올바르지 않습니다 메시지가 표시됩니다.

ℹ️
비밀번호를 분실한 경우 관리자에게 초기화를 요청하세요. 사용자 관리 화면에서 비밀번호 재설정 필드로 즉시 처리할 수 있습니다.

2생산요청

관리자 영업

생산할 충전기의 모델·용도·수량만 입력해 PENDING 오더를 등록하는 화면입니다. 발주처·납품일·설치주소 등 상세 정보는 이 단계에서 입력하지 않으며, 생산 대기 화면의 생산완료 단계에서 입력합니다.

행 입력

한 행이 모델·용도 조합 하나를 의미하며, 여러 행을 동시에 등록할 수 있습니다.

항목설명
충전속도(모델) *모델 선택... 버튼으로 팝업을 열어 충전속도 칩을 먼저 고른 뒤 해당 속도의 모델 목록에서 선택
충전기 용도공용 / 비공용 칩 토글 (기본값 공용, 새 행 추가 시 직전 행의 용도를 이어받음)
수량−/+ 스테퍼 또는 직접 입력, 1~500 범위로 자동 보정

행 추가 / 복제 / 삭제

  • 표 하단의 개수 입력(1~50) + + 행 추가 버튼으로 한 번에 여러 행을 추가
  • 행의 복제 아이콘으로 같은 모델·용도의 행을 바로 아래에 추가
  • 행의 삭제(×) 아이콘으로 개별 행 삭제 (마지막 한 행은 삭제 버튼이 숨겨짐)

다중 선택 일괄 처리

행 왼쪽 체크박스로 여러 행을 선택하면 툴바가 나타나며 모델 일괄 지정, 공용/비공용 일괄 지정, 선택 삭제, 선택 해제를 한 번에 적용할 수 있습니다.

등록

화면 상단에 {건수}건 등록 예정 요약이 실시간으로 표시됩니다. {n}건 생산요청 등록 버튼을 누르면 각 행이 수량만큼 반복 등록되어 생산 대기 목록에 추가됩니다. 등록 시 requested_by에 로그인한 사용자 이름이 함께 저장됩니다.

⚠️
등록된 오더는 PENDING 상태일 때만 수정할 수 있으며, 모델과 용도만 고칠 수 있습니다. 목록에서 카드를 클릭하면(영업 역할만) 수정 화면이 열립니다.

마스터 데이터 관리

고객사·담당자·CPO 운영사는 생산완료 단계에서 입력하지만, 목록에 없는 값은 해당 입력창의 + 추가 버튼으로 바로 등록할 수 있습니다. 관리자는 이 화면 상단 모델 / 모델 관리 버튼으로 모델 마스터를 직접 추가·수정할 수 있습니다.

3생산 대기

관리자 영업 생산

모든 진행 중인 오더를 칸반 보드 하나로 관리하는 화면입니다. 카드를 클릭하면 역할과 컬럼에 따라 서로 다른 입력창이 열립니다.

칸반 컬럼

컬럼오더 상태카드 클릭 시 (해당 권한)
생산요청PENDING영업 — 모델·용도 수정 화면
생산착수IN_PROGRESS생산 — 생산착수 모달
생산완료AWAIT_PICKUP (영업정보 미입력)영업 — 생산완료 모달
출하대기AWAIT_PICKUP (영업정보 입력 완료)출하대기 화면 접근 권한이 있는 역할 — 출하대기 화면으로 이동

생산착수·생산완료·출하대기 컬럼은 해당하는 오더가 없으면 화면에서 자동으로 숨겨집니다.

검색 · 필터 · 신규 생산요청

  • 상단 검색창에서 고객사·모델·충전소 ID·오더번호로 검색
  • 모델 드롭다운 필터, 비공용만 칩 토글, 초기화 버튼
  • 우측 상단 신규 생산요청 버튼으로 생산요청 화면으로 바로 이동

다중 선택 일괄 생산착수

생산요청 컬럼(영업 역할 제외)에서 카드 체크박스로 여러 건을 선택하면 생산 착수 ({n}건) 버튼이 나타납니다. 클릭 시 확인창(생산대기 {n}건을 생산착수로 전환할까요?) 후 한 번에 IN_PROGRESS로 전환되며, 생산 실적(시리얼 등)은 이후 개별 카드를 눌러 입력합니다.

D-day 배지

출하대기 컬럼 카드에만 납품일 기준 D-day 배지가 표시됩니다.

  • 날짜 미정 — 회색 일정 미정
  • 납품일 경과 또는 D-7 이내 — 빨간색 D+n / D-n
  • D-8 ~ D-14 — 주황색 D-n
  • D-15 이상 — 회색 D-n

생산착수 모달 (생산 · 관리자)

생산착수 컬럼 카드를 클릭하면 열립니다. PENDING → IN_PROGRESS 전환 시 시리얼이 이미 자동 채번되어 있으며, 이 화면에서 나머지 생산 실적을 입력합니다.

자주 묻는 질문

Q. 생산완료와 출하대기는 어떻게 다른가요?

둘 다 상태값은 AWAIT_PICKUP으로 동일합니다. 발주처·납품정보(영업정보) 입력이 끝나지 않았으면 생산완료 컬럼에, 끝났으면 출하대기 컬럼과 화면에 나타납니다.

Q. 여러 건을 한 번에 생산착수할 수 있나요?

네. 생산 대기 화면의 생산요청 컬럼에서 카드를 다중 선택한 뒤 생산 착수 ({n}건) 버튼을 누르면 한 번에 IN_PROGRESS로 전환됩니다. 시리얼 등 생산 실적은 이후 개별 카드에서 입력합니다.

Q. 오더 등록 후 수정이 가능한가요?

영업 역할은 오더가 PENDING 상태일 때만 모델·용도를 수정할 수 있습니다. 발주처·납품정보는 생산완료 모달에서 저장 후에도 다시 열어 수정할 수 있습니다.

Q. 시리얼 번호는 언제, 어떻게 채번되나요?

생산 착수 시점(PENDING → IN_PROGRESS 전환 직후)에 모델 코드 + 생산 시점 기반 채번 규칙으로 자동 채번됩니다. 생산착수 모달에서 재생성 링크로 다시 채번하거나 직접 수정할 수 있으며, 중복 확인 버튼으로 검증합니다.

Q. 기능·출하 검사 체크리스트는 어디서 관리하나요?

docs/func/{모델코드}.json(기능검사), docs/ship/{모델코드}.json(출하검사) 파일로 모델별 체크리스트를 관리합니다. 파일이 없으면 시스템 기본 체크리스트가 사용됩니다.

Q. 출하 전 사진은 어떻게 첨부하나요?

출하대기 목록에서 출하검사 성적서를 먼저 저장한 뒤, 드로어 하단의 사진 첨부 탭에서 파일을 선택하면 Supabase Storage(ship-photos 버킷)에 자동 업로드됩니다.

Q. CSV 내보내기 시 한국어가 깨집니다.

파일을 Excel에서 열 때는 데이터 → 텍스트/CSV에서 가져오기를 사용하고 인코딩을 UTF-8로 선택하세요. 파일을 더블클릭해서 바로 열면 인코딩이 자동으로 인식되지 않을 수 있습니다.

Q. 비밀번호를 잊었을 때는?

관리자에게 비밀번호 초기화를 요청하세요. 사용자 관리 화면에서 비밀번호 재설정 필드로 즉시 처리할 수 있습니다.
